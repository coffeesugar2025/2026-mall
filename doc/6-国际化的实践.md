# SpringBoot + Vue3 前后端分离国际化完整整合方案
## 整体架构分工
1. **Vue3(vue-i18n)**：页面菜单、按钮、表单、组件静态文本渲染
2. **SpringBoot(MessageSource)**：接口返回提示、异常文案、后端参数校验信息
3. 语言同步规则：前端切换语言后，通过 `lang` 请求头同步至后端，全局语言环境统一
4. 语言编码统一规范：`zh_CN` / `en_US`（与现有SpringBoot资源文件名完全对齐）

## 2. I18n核心配置类（优先读取Header lang，兼容URL参数）
```java
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.web.servlet.LocaleResolver;
import org.springframework.web.servlet.config.annotation.InterceptorRegistry;
import org.springframework.web.servlet.config.annotation.WebMvcConfigurer;
import org.springframework.web.servlet.i18n.LocaleChangeInterceptor;
import org.springframework.web.servlet.i18n.SessionLocaleResolver;
import javax.servlet.http.HttpServletRequest;
import java.util.Locale;

@Configuration
public class I18nConfig implements WebMvcConfigurer {

    /**
     * 自定义语言解析器：优先读取请求头lang，其次URL参数?lang=xxx
     */
    @Bean
    public LocaleResolver localeResolver() {
        return new SessionLocaleResolver() {
            @Override
            public Locale resolveLocale(HttpServletRequest request) {
                // 1. 优先读取前端axios携带的lang请求头
                String headerLang = request.getHeader("lang");
                if (headerLang != null && !headerLang.isEmpty()) {
                    String[] split = headerLang.split("_");
                    return new Locale(split[0], split[1]);
                }
                // 2. 兼容Postman测试URL参数 lang=en_US
                String paramLang = request.getParameter("lang");
                if (paramLang != null && !paramLang.isEmpty()) {
                    String[] split = paramLang.split("_");
                    return new Locale(split[0], split[1]);
                }
                // 默认中文环境
                return Locale.CHINA;
            }
        };
    }

    /**
     * 拦截器捕获URL lang参数切换语言
     */
    @Bean
    public LocaleChangeInterceptor localeChangeInterceptor() {
        LocaleChangeInterceptor interceptor = new LocaleChangeInterceptor();
        interceptor.setParamName("lang");
        return interceptor;
    }

    @Override
    public void addInterceptors(InterceptorRegistry registry) {
        registry.addInterceptor(localeChangeInterceptor());
    }
}
```

## 3. 全局i18n工具类（任意位置快速获取翻译）
```java
import org.springframework.context.MessageSource;
import org.springframework.context.i18n.LocaleContextHolder;
import org.springframework.stereotype.Component;
import javax.annotation.Resource;

@Component
public class MsgUtil {
    @Resource
    private MessageSource messageSource;

    /**
     * 获取国际化翻译文案
     * @param code 翻译key
     * @param args 占位符参数
     * @return 对应语言文案
     */
    public String get(String code, Object... args) {
        return messageSource.getMessage(code, args, LocaleContextHolder.getLocale());
    }
}
```

## 4. 国际化测试 Controller
```java
import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.context.MessageSource;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestParam;
import org.springframework.web.bind.annotation.RestController;
import java.util.Locale;

@Slf4j
@RestController
public class I18nController {

    @Autowired
    private MessageSource messageSource;

    @GetMapping("/hello")
    public String hello(@RequestParam String name) {
        Locale locale = LocaleContextHolder.getLocale();
        // 第四个参数：兜底默认文案，缺失key不会抛出400异常
        return messageSource.getMessage("greeting", new Object[]{name}, "默认问候", locale);
    }
}
```

## 4. locales/index.ts i18n初始化与语言切换工具
```typescript
import { createI18n, useI18n } from 'vue-i18n'
import zhCN from './zh_CN'
import enUS from './en_US'

// 读取本地存储的上次选择语言，默认中文
const localLang = localStorage.getItem('lang') || 'zh_CN'

const i18n = createI18n({
  legacy: false, // 关闭Vue2兼容，启用组合式API
  locale: localLang,
  fallbackLocale: 'zh_CN', // 翻译缺失时降级中文
  messages: {
    zh_CN: zhCN,
    en_US: enUS
  }
})

/**
 * 语言切换全局钩子
 */
export const useLangSwitch = () => {
  const { locale, t } = useI18n()
  const switchLang = (lang: 'zh_CN' | 'en_US') => {
    // 更新前端页面语言
    locale.value = lang
    localStorage.setItem('lang', lang)
    // 同步更新axios全局请求头，后端自动识别语言
    import('@/utils/request').then(({ service }) => {
      service.defaults.headers.common['lang'] = lang
    })
  }
  return { locale, t, switchLang }
}

export default i18n
```

## 6. axios 请求拦截器（核心：同步lang至后端）
```typescript
// utils/request.ts
import axios from 'axios'

const service = axios.create({
  baseURL: 'http://localhost:8000',
  timeout: 15000
})

// 请求拦截：每次接口自动携带lang请求头
service.interceptors.request.use(config => {
  const lang = localStorage.getItem('lang') || 'zh_CN'
  config.headers.lang = lang
  return config
})

// 响应拦截：后端错误码可结合i18n翻译提示
service.interceptors.response.use(res => {
  return res.data
}, err => {
  return Promise.reject(err)
})

export default service
```

  const data = await request.get('/hello?name=Tom')
  resText.value = data

})

# 三、两种整合方案对比（项目选型参考）
## 方案1：前后端翻译分离（推荐90%业务项目）
| 模块                     | 职责                                   | 优势                           |
| ------------------------ | -------------------------------------- | ------------------------------ |
| Vue3 i18n                | 页面菜单、按钮、静态提示               | 修改页面文本无需重启后端，解耦 |
| SpringBoot MessageSource | 接口返回、异常、校验文案               | 后端统一管控错误提示、接口返回 |
| 同步方式                 | 前端切换语言 → axios携带 `lang` 请求头 | 职责清晰，维护成本低           |

## 方案2：共用一套翻译源（大型多语言平台）
1. 后端开发接口读取 `messages` 配置文件，返回JSON格式语言包
2. Vue初始化时异步请求后端接口加载全部翻译，前后端共用同一批key
3. 仅维护一份properties文件，无需同步前后端翻译文件
- 优点：只维护一套翻译，避免key不一致
- 缺点：前端初始化依赖后端接口，离线页面无法正常展示

# 四、常见报错&问题排查
## 1. 后端报错 `No message found under code 'greeting' for locale 'en_US'`
1. 确认 `spring.messages.basename=i18n/messages` 配置正确
2. `messages_en_US.properties` 文件内存在 `greeting=Hello {0}`
3. 清理项目 `target` 目录，重启SpringBoot服务

## 2. 前端切换语言，后端返回文案不跟随切换
1. 检查axios请求拦截器是否正常设置 `headers.lang`
2. 确认后端 `LocaleResolver` 优先读取Header中的lang字段
3. 清除浏览器localStorage，重新切换语言测试

## 3. 中文翻译出现乱码
1. IDEA全局文件编码全部设置为 `UTF-8`，勾选 `Transparent native-to-ascii conversion`
2. yml配置中指定 `spring.messages.encoding=UTF-8`

## 4. Element Plus / Ant Design Vue UI组件语言不同步
在 `switchLang` 方法内同步UI组件库语言：
```typescript
import { ElLocale } from 'element-plus'
import zhCn from 'element-plus/es/locale/lang/zh-cn'
import en from 'element-plus/es/locale/lang/en'

const switchLang = (lang: 'zh_CN' | 'en_US') => {
  locale.value = lang
  localStorage.setItem('lang', lang)
  // 同步UI组件语言
  ElLocale.value = lang === 'zh_CN' ? zhCn : en
  import('@/utils/request').then(({ service }) => {
    service.defaults.headers.common['lang'] = lang
  })
}
```

2. 前端所有接口请求自动携带请求头 `lang: en_US`
