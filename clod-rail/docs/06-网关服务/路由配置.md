# 网关路由配置

## 路由规则

| 服务 | 路由前缀 | 目标服务 |
|------|----------|----------|
| 用户服务 | `/customer/user/**` | user-service |
| 用户服务 | `/customer/contact/**` | user-service |
| 用户服务 | `/customer/auth/**` | user-service |
| 用户服务 | `/admin/auth/**` | user-service |
| 用户服务 | `/admin/user/**` | user-service |
| 车票服务 | `/customer/tickets/**` | ticket-service |
| 车票服务 | `/customer/stations/**` | ticket-service |
| 车票服务 | `/admin/tickets/**` | ticket-service |
| 车票服务 | `/admin/stations/**` | ticket-service |
| 车票服务 | `/admin/lines/**` | ticket-service |
| 车票服务 | `/admin/trains/**` | ticket-service |
| 订单服务 | `/customer/orders/**` | order-service |
| 订单服务 | `/customer/pay/**` | order-service |
| 订单服务 | `/admin/orders/**` | order-service |
| 商城服务 | `/customer/points/**` | mall-service |
| 商城服务 | `/customer/mall/**` | mall-service |
| 商城服务 | `/admin/mall/**` | mall-service |
| 客服服务 | `/customer/assistant/**` | assistant-service |
| 客服服务 | `/admin/memory/**` | assistant-service |

## 路由前缀说明

| 前缀 | 说明 |
|------|------|
| `/customer/**` | 用户端接口 |
| `/admin/**` | 管理端接口 |
| `/inner/**` | 内部服务调用 |
| `/common/**` | 公共接口 |

## 配置示例

```yaml
spring:
  cloud:
    gateway:
      routes:
        - id: user-service
          uri: lb://user-service
          predicates:
            - Path=/customer/user/**, /customer/auth/**, /admin/auth/**
        - id: ticket-service
          uri: lb://ticket-service
          predicates:
            - Path=/customer/tickets/**, /customer/stations/**
```
