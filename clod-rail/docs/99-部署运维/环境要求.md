# 环境要求

> 版本号以本项目实际依赖为准。若你用更低版本运行,请自行承担兼容性风险。

## 开发环境

| 软件 | 版本 | 说明 |
|------|------|------|
| JDK | **17+** | 项目 `pom.xml` 声明 `java.version=17`,也可运行在 21 上 |
| Maven | 3.8+ | 构建工具 |
| Node.js | 18+ | 前端构建 |
| MySQL | 8.0+ | 数据库 |
| Redis | 7.0+ | 缓存 / 分布式锁 / Bitmap |
| RabbitMQ | 3.12+ | 消息队列 |
| Nacos | 2.2+ | 注册中心 + 配置中心 |

## 框架版本(已在 `pom.xml` 锁定,无需手动配置)

| 依赖 | 版本 |
|------|------|
| Spring Boot | **3.5.5** |
| Spring Cloud | **2025.0.0** |
| Spring Cloud Alibaba | **2023.0.3.2** |
| Seata Spring Boot Starter | **2.3.0**(`org.apache.seata`,Apache 顶级项目包名) |
| MyBatis Plus、Hutool 等 | 见 `rs-util/*` 各 starter |

## 可选组件

| 软件 | 版本 | 说明 |
|------|------|------|
| Elasticsearch | 8.x | 商城搜索,未启用时降级 MySQL LIKE |
| **Seata Server** | **2.3.x(Apache 版本)** | 订单 TCC 分布式事务。**注意**:项目用的是 `org.apache.seata` 包(2.x 捐给 Apache 后的坐标),请下载 Seata 2.x 服务端,而非 1.x |
| XXL-JOB Admin | 2.4+ | 任务调度(订单超时关单、热点预热) |

## IDE 推荐

- IntelliJ IDEA（推荐旗舰版）
- VS Code（前端开发）

## 工具推荐

| 工具 | 用途 |
|------|------|
| Postman / Apifox | API 测试 |
| Navicat / DataGrip | 数据库管理 |
| RedisInsight | Redis 管理 |
| Git | 版本控制 |
