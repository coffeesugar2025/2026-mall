## 快速了解

ClodRail 是一个仿 12306 的铁路票务微服务系统,包含:

- 🧑‍💼 用户服务 - 注册、登录(Access + Refresh 双 Token)、权限管理
- 🎫 车票服务 - 车站、线路、车次、余票
- 📦 订单服务 - 下单、支付、退票(Seata TCC)
- 🛒 积分商城 - 积分、商品兑换、ES 全文检索
- 🤖 客服助手 - LangChain4j AI 问答、Netty WebSocket
- 🚪 统一网关 - 路由、鉴权、限流、RBAC
